## 📌 Sobre esse repositório
O Frontend Mentor é uma plataforma online com objetivo em ajudar desenvolvedores front-end a aprimorar suas habilidades que disponibiliza desafios simulando práticas reais. Esses desafios utilizam linguagens como HTML, CSS, JavaScript, React e etc.

Esse repo é onde deixo registrado todas as minhas soluções dos desafios além de compartilhar um pouco do processo enquanto desenvolvia cada um.
