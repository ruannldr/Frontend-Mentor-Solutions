## 📌 About this repository

Frontend Mentor is an online platform aimed at helping front-end developers improve their skills by providing challenges that simulate real-world practices. These challenges use languages such as HTML, CSS, JavaScript, React, etc.

This repo is where I keep track of all my challenge solutions while also sharing a bit of the process as I developed each one.
